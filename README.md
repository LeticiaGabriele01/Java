# 🎓 Aulas de Programação - Faculdade 2026

Bem-vindo ao meu repositório de estudos! Aqui organizo os códigos desenvolvidos durante as aulas de computação, focando na aplicação prática dos conceitos aprendidos.

## 📚 Matérias em Destaque

Neste semestre, o foco está em dominar a linguagem Java através de duas disciplinas fundamentais:

### ☕ Programação Orientada a Objetos (POO)
Estudo dos pilares da orientação a objetos para criar sistemas modulares, reutilizáveis e de fácil manutenção.
* **Conceitos:** Abstração, Encapsulamento, Herança e Polimorfismo.
* **Prática:** Criação de classes, interfaces e tratamento de exceções.

### 🏗️ Estrutura de Dados II
Exploração de algoritmos avançados e formas eficientes de organizar e manipular grandes volumes de dados.
* **Conceitos:** Árvores (Binárias, AVL, B), Grafos, Tabelas Hash e Algoritmos de Ordenação avançados.
* **Prática:** Implementação de estruturas dinâmicas e análise de complexidade.
